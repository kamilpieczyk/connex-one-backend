Node.js, Express.js, TypeScript

ts-node "^10.5.0",
typescript "^4.5.5"

# How to run

first install all required dependencies: 

```shell
npm install
```

then run with script

```shel
npm run start
```

or with nodemon

```shel
npm run start:dev
```
